# POS-система для малого бизнеса

Готовая к запуску система автоматизации кассы и товарного учёта для небольших магазинов, киосков и кофеен.

## Быстрый старт

### Windows
```
Двойной клик на start.bat
```

### Linux / macOS
```bash
chmod +x start.sh
./start.sh
```

### Вручную
```bash
pip install -r requirements.txt
python app.py
```

После запуска браузер откроется автоматически: **http://127.0.0.1:5000**

## Учётные записи по умолчанию

| Роль | Логин | Пароль |
|------|-------|--------|
| Администратор | `admin` | `admin` |
| Кассир | `кассир` | `1234` |

**Обязательно смените пароли!** Настройки → Пользователи

## Функциональность

### Касса (рабочее место кассира)
- Поиск товара по названию, артикулу или штрихкоду
- Поддержка USB-сканеров штрихкодов (HID-режим)
- Быстрые кнопки для часто продаваемых товаров (до 12 шт.)
- Оплата наличными, картой или смешанная
- Автоматический расчёт сдачи
- Печать HTML-чека (открывается в браузере)
- Карты лояльности со скидкой
- Скидка на чек (% или фиксированная сумма)

### Товарный учёт
- Справочник товаров с категориями
- Учёт остатков на складе
- Предупреждение при низком остатке
- Приходные накладные и акты списания
- Импорт товаров из CSV

### Управление сменами
- Открытие/закрытие смены
- X-отчёт (промежуточный)
- Z-отчёт (итоговый при закрытии)
- Учёт расхождения фактического и ожидаемого остатка

### Отчёты (только администратор)
- Продажи по дням
- Выручка по кассирам
- Продажи по товарам (с прибылью)
- Текущие остатки склада
- Экспорт всех отчётов в CSV

### Администрирование
- Управление пользователями (роли: администратор, кассир)
- Настройки магазина (название, ИНН, адрес)
- Резервное копирование и восстановление базы данных

## Структура файлов

```

├── app.py              # Основное Flask-приложение
├── pos.db              # База данных SQLite (создаётся автоматически)
├── pos.log             # Лог действий
├── requirements.txt    # Зависимости Python
├── start.bat           # Запуск Windows
├── start.sh            # Запуск Linux/macOS
├── templates/          # HTML-шаблоны
│   ├── base.html       # Базовый шаблон с меню
│   ├── login.html      # Страница входа
│   ├── cashier.html    # Рабочее место кассира
│   ├── products.html   # Справочник товаров
│   ├── categories.html # Категории
│   ├── inventory.html  # Движение товаров
│   ├── sales_history.html # Журнал продаж
│   ├── reports.html    # Отчёты
│   ├── settings.html   # Настройки
│   ├── users.html      # Пользователи
│   └── receipt.html    # Шаблон чека
└── static/
    ├── style.css       # Стили
    ├── app.js          # Общий JS
    └── cashier.js      # Логика кассы
```

## Требования

- Python 3.10+
- Flask 3.0+
- Современный браузер (Chrome, Firefox, Edge)
- Минимальный экран: 1024×768
- Работает полностью офлайн

## Сканер штрихкодов

Поддерживаются USB HID-сканеры (работают как клавиатура). При сканировании штрихкод автоматически вводится в поле поиска и товар добавляется в корзину.

## Резервное копирование

Вся информация хранится в файле `pos.db`. Для резервного копирования:
- **Из программы**: Настройки → «Скачать резервную копию»
- **Вручную**: скопируйте файл `pos.db` на флешку
